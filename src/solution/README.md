# 方案

* 汽车安全方案
  * 毕马威+ESCRYPT
    * 优势和经验
      * `毕马威`：`信息安全审计`
      * `ESCRYPT`：`汽车安全工程`
    * CSMS解决方案
      * `PROOF`=`Product Security Organization Framework`=`产品安全组织框架`
        * 将网络安全要求集成到企业管理和汽车工程之中，PROOF将有助于企业有效集中资源，快速制定和实施满足业务发展并且行业合规的CSMS体系和流程，建立并形成涵盖企业端和产品端的网络安全持续改进能力
        * 白皮书：汽车网络安全白皮书 (assets.kpmg)
